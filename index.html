기본 구조 태그

HTML 문서의 시작과 끝을 나타냅니다
HEAD 문서의 메타데이터(제목, 스타일, 스크립트 등)를 포함합니다
TITLE 웹 브라우저의 탭 제목을 설정합니다.
BODY 실제 웹 페이지의 내용을 포함하는 부분입니다.

텍스트 관련 태그

H1 ~ H6 제목을 나타내는 태그로, 숫자가 작을수록 글자가 큽니다.
(<h1>이 가장 크고 <h6>이 가장 작음)
P 단락(문단)을 만들 때 사용합니다.
BR 줄바꿈을 할 때 사용합니다. (<br>은 종료 태그 없이 사용)
HR 수평선을 표시합니다.
STRONG 텍스트를 강조(굵게)합니다.
EM 텍스트를 강조(기울임)합니다.

링크와 이미지 관련 태그

A 하이퍼링크를 생성합니다.
예: <a href="https://www.google.com">Google</a>
IMG 이미지를 삽입합니다.
예: <img src="image.jpg" alt="설명 텍스트">

목록 관련 태그

UL 순서 없는 목록(●, ○ 등)을 만듭니다.
OL 순서 있는 목록(1, 2, 3 등)을 만듭니다.
LI 목록 항목을 나타냅니다.

테이블 관련 태그

TABLE 테이블(표)을 생성합니다.
TR 테이블 행을 나타냅니다.
TD 테이블의 개별 셀을 나타냅니다.
TH 테이블 헤더 셀을 나타냅니다.

폼 관련 태그

FORM 사용자가 데이터를 입력하고 제출할 수 있는 폼을 만듭니다.
INPUT 사용자가 입력할 수 있는 입력 필드를 생성합니다.
TEXTAREA 여러 줄의 텍스트를 입력할 수 있는 입력 상자입니다.
BUTTON 버튼을 생성합니다.

시맨틱 태그

HEADER 머리말(로고, 내비게이션 등)을 포함하는 영역.
NAV 내비게이션 메뉴를 나타냅니다.
SECTION 페이지의 주요 섹션을 나타냅니다.
ARTICLE 독립적인 콘텐츠(게시글, 기사 등)를 나타냅니다.
FOOTER 페이지의 하단(저작권, 연락처 등)을 나타냅니다.
